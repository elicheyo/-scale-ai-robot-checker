# Scale AI Robot Fleet Status Checker 🤖

A Python diagnostic tool that checks if a robot 
passes or needs escalation based on accuracy score.

## What it does
- Takes robot ID, accuracy score, and online status
- Checks if accuracy is at or above 90%
- Prints PASS or ESCALATE status

## Built with
- Python 3
- input() — user input
- Type casting — float()
- if/else logic
- f-strings

## Example output
✅ PASS | Robot: TROSSEN-01 | Accuracy: 93.3% | Online: yes
⚠️ ESCALATE | Robot: TROSSEN-07 | Accuracy: 78.5% | Online: NO
